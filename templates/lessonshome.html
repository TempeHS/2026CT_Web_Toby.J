Beginner Lessons

Indentation

Indentation in Python defines the structure of the code by marking blocks of statements. It replaces traditional braces and ensures readability.

Variables

A variable in Python is a named reference to a value, allowing data storage and reuse. It can hold numbers, text, or other types.

Operators

Basic Python operators include addition, subtraction, multiplication, and division. These operators are used to perform arithmetic operations on numbers.


Intermediate Lessons

Booleans

In Python, booleans are a fundamental data type representing truth values, with two possible values: True or False. They are essential for controlling program flow, making decisions, and performing comparisons.

Variable Types

Python's variables are containers for storing data values. These values can be of various types, each with its own characteristics and usage. Some examples include integers, floats, strings and booleans, which each produce different values. This lesson is a breakdown of common variable types in Python.

Formatting Strings

String formatting in Python allows you to embed variables and expressions within strings. There are several methods to achieve this, each with its syntax and features. Some examples can include slicing, concatenating and modifying. Some examples of commands can include uppercase, lowercase, splitting and escape characters. They are all covered in this lesson.
